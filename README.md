# Usage

1. Clone and cd

```bash
git clone https://github.com/nakamarusun/othello-minimax.git othello
cd othello
```

2. Compile and run (Make sure g++ is installed)

```bash
# Powershell
g++ src/*.cpp -o game ; ./game

# Mac & Linux
g++ src/*.cpp -o game && ./game

# Cmd
g++ src\*.cpp -o game && .\game
```
